# Product Manager Spec

## Overview

Task Flow API is a task management REST API that enables users to organize work into named lists, each containing prioritized tasks with lifecycle status tracking. The system provides JWT-based authentication, full CRUD for lists and tasks, and strict user-scoped access control so each user only sees their own data. Built for demonstration as the LightSpec brownfield example, it targets individual users and small teams needing a lightweight, self-hosted task tracker. Success is measured by a clean separation of routing, service, and persistence layers with 100% integration test coverage of all endpoints.
